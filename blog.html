Question-3: What is event delegation in the DOM?

Event Delegation is a technique where, instead of adding event handlers directly to child elements, we add the event handler to their parent or an ancestor element to capture events from the child elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a behavior of DOM events where, when an event like a click happens on a child element, the event first triggers on the child, then propagates to its parent element, and then continues to bubble up to the top all the way to the root element - document.
